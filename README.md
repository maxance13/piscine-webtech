
Piscine Webtech N2 Projet-Django

"Une application web Django pour rechercher des films et afficher des détails via l'API TMDB, avec une fonctionnalité de blog,avec systeme de login, de message et de contact."

---------------------------------------------------------------------------------------------------

Fonctionnalités : 

Recherche de films via l'API TMDB.
Affichage de détails de films.
Section de blog avec des articles et des commentaires.

---------------------------------------------------------------------------------------------------

Prérequis
Ce projet nécessite Python 3.x et Django 3.x. 
Assurez-vous que Python et Django sont installés sur votre système.

Installation
Pour configurer ce projet sur votre système local, 
suivez les étapes ci-dessous :

Cloner le Répertoire :

bash
Copy code
git clone https://github.com/maxance13/piscine-webtech.git
cd votre-projet-django
Configurer l'Environnement Virtuel :

bash
Copy code
python -m venv venv
source venv/bin/activate  # Sur Windows utilisez `venv\Scripts\activate`
Installer les Dépendances :

bash
Copy code
pip install -r requirements.txt
Configuration des Variables d'Environnement :

Créez un fichier .env dans le répertoire principal du projet 
et ajoutez-y les variables d'environnement nécessaires 
(par exemple, les clés API).

--------------------------------------------------------------------------------------------------

Initialiser la Base de Données :

bash
Copy code
python manage.py makemigrations
python manage.py migrate
Créer un Superutilisateur :

bash
Copy code
python manage.py createsuperuser
Lancer le Serveur :

bash
Copy code
python manage.py runserver
Visitez http://127.0.0.1:8000 dans votre navigateur.
